# be-study-2026
백엔드 학습

## 10주 과정(week1 ~ week10)
| 단계 | 주차 | 핵심 내용 | 획득 스킬 |
|------|------|-----------|------------|
| **Foundation** | 1~2주 | 언어와 웹 | Java, Spring Boot, HTTP, MVC, REST API |
| **Core** | 3~4주 | DB와 보안 | JPA, Entity 설계, Spring Security, 인증/인가 |
| **Architecture** | 5~6주 | 동시성 & 결제 | 비관적 락(Lock), 외부 API 연동, 트랜잭션 관리 |
| **Quality** | 7주 | 테스트(TDD) | JUnit 5, 멀티스레드 테스트, 테스트 격리 전략 |
| **Performance** | 8~9주 | 최적화 & 쿼리 | N+1 문제 해결(Fetch Join), QueryDSL, 동적 쿼리 |
| **Operation** | 10주 | 안정성 & AOP | 주문 취소(데이터 무결성), AOP(모니터링) |
